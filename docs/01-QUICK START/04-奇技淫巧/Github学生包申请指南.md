# Github学生包申请指南

Github学生包是什么?

GitHub 学生包（GitHub Student Developer Pack）是 GitHub 面向学生提供的一系列免费开发工具、服务和资源合集。你可以免费或大幅折扣地使用一些本来需要付费的开发工具和服务, 就比如满血版Copilot(强大的代码辅助AI), 还有就是Azure服务器(可以拿来搭建个人博客, 代理等)。
