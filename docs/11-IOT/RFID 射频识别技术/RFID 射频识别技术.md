# RFID 射频识别技术

## 常见的 RFID 设备

- [FillerZero](https://flipper.net/) (功能较多, 价格高, 但不够专业)
- [Proxmark3](https://github.com/Proxmark/proxmark3) (专业的 RFID 设备, 价格较高)
- [ChameleonUltra](https://github.com/RfidResearchGroup/ChameleonUltra) (性价比较高, 基本功能齐全)

## 常见的 RFID 软件 

- [NFC Tools](https://play.google.com/store/apps/details?id=com.wakdev.wdnfc&hl=en&gl=US) (Android 上的 NFC 工具, 可以读取和写入 NFC 标签)
- [MifareClassicTool](https://github.com/ikarus23/MifareClassicTool)
